# PROJECT.md — Методология Созидания

## Суть проекта

Идейный академический портал о мировоззрении, методологии, ДОТУ, ВП СССР и Экосистеме Созидания. Наследие В.А. Ефимова. Подход: не от терминов — от жизненных вопросов к идеям.

Целевая аудитория: новички, которые хотят разобраться в системном мышлении, обществе, семье, управлении, культуре.

## Стек (утверждён)

**HTML-файлы** — без фреймворков на первом этапе.  
Позже: миграция на Astro SSG, когда сайт вырастет до 30+ страниц.  
Хостинг: GitHub Pages (бесплатно).

## Состояние

- Главная страница: standalone HTML с base64-изображениями (~660 КБ) — **ПРОТОТИП, требует доработки**
- Все остальные страницы: **НЕ НАЧАТЫ**

## Приоритеты

### Фаза 1 — Доработка главной страницы
1. Переключатель тем (свет/тьма, старт всегда со светлой)
2. Текстовые правки (ждём от пользователя)
3. Замена 10 изображений на русский стиль (матрёшки, берёзы, иконопись, орнаменты)
4. Кнопки шеринга (ВК, Дзен, RuTube)
5. Вынести изображения из base64 в файлы

### Фаза 2 — Внутренние страницы
- /about-worldview — О мировоззрении
- /topics/* — 6 тематических страниц
- /concepts — Основные понятия
- /start — С чего начать
- /library — Библиотека
- /analysis — Разборы
- /school — Обучение
- /faq — Вопросы и ответы
- /contacts — Контакты

### Фаза 3 — Инфраструктура
- Переезд на Astro SSG
- SEO-оптимизация
- Оптимизация изображений (WebP)
- Настройка GitHub Pages

## Дизайн-система

Префикс классов: `ms-`  
Темы: светлая (по умолчанию) + тёмная (по переключателю)  
Типографика: Cormorant Garamond (заголовки) + Inter (текст)  
Акценты: gold (#C9A84C), green (#3A7D5C)

## Ключевые ограничения

- Не выдумывать факты о ДОТУ, ВП СССР, Ефимове
- Не использовать слова «тайные знания», «пробуждение», «избранные»
- Не культ личности — работа с идеями
- Ясность для новичка важнее академической сложности

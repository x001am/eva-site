## Что такое проект

Сайт для новичков, которые хотят разобраться в системном мышлении, устройстве общества, семьи, управления, культуры. Подход: не от терминов к человеку, а от жизненных вопросов — к идеям и методологии.

Это НЕ про культ личности. Это про идеи.

Тон: академический, спокойный, серьёзный, ясный. Без эзотерики, без «тайных знаний», без рекламности. Если информации нет — говорим прямо.

## Дизайн-система

### Палитра

```css
--ms-bg: #0C0F0E;                        /* основной фон */
--ms-surface: #1A1F1E;                   /* фон карточек */
--ms-cream: #F5F0E8;                     /* текст основной */
--ms-cream-f: rgba(245,240,232,0.7);     /* текст вторичный */
--ms-gold: #C9A84C;                      /* акцент основной */
--ms-gold-d: #A8893A;                    /* акцент тёмный */
--ms-green: #3A7D5C;                     /* акцент вторичный */
--ms-border: rgba(245,240,232,0.08);     /* бордюры */
--ms-border-h: rgba(245,240,232,0.15);   /* бордюры hover */
```

### Типографика

- Заголовки: `Cormorant Garamond` (serif), font-weight: 400–500, акценты — italic + золотой
- Текст: `Inter` (sans-serif), font-weight: 300/400/500
- Размеры заголовков через `clamp()` для адаптивности

### Визуальный стиль

Тёмная кинематографичная тема. Вдохновение: Home of the Brave, Dagbladet, Tremaine Collection, Harvard Century-Scale Storage.

### CSS-архитектура

Все классы с префиксом `ms-` (предотвращает конфликты при интеграции в CMS/Тильду):
- `.ms-nav`, `.ms-hero`, `.ms-qcard`, `.ms-tcard`, `.ms-step` и т.д.
- `.ms-reveal` + `.ms-visible` — для scroll-анимаций
- `!important` на цвета кнопок и ссылок (нужен для Тильды, в standalone можно убрать)

### Реализованные эффекты

1. **3D Tilt карточек** — perspective(1000px), rotateX/Y ±6°, реакция на мышь
2. **Glow** — radial-gradient(600px circle) следует за курсором. Золотой (#C9A84C) для вопросов, зелёный (#3A7D5C) для тем
3. **Частицы на Hero** — золотые точки, setInterval(400ms), живут 20с, удаляются автоматически
4. **Параллакс Hero** — только фон двигается (20px), контент статичен
5. **Scroll Reveal** — IntersectionObserver, threshold 0.1, translateY(30px) → 0
6. **Hover на изображениях** — brightness 0.65→0.85, scale(1.03–1.04)
7. **Пульсирующая линия «листайте»** — keyframes scaleY + opacity
